# Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**.  
This website is designed to showcase my **profile, skills, projects, certifications, resume, and contact information** in a professional and user-friendly way.

---

## 📌 Project Overview

This portfolio website represents my personal and professional profile on the web. It helps visitors understand who I am, what skills I have, the projects I have completed, and how to contact me. The website is fully responsive and provides a clean user experience across different devices.

---

## 🚀 Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and attractive user interface
- Dark and light theme toggle
- Smooth scrolling navigation
- Active section highlight in navigation bar
- Animated content reveal on scroll
- Project filtering option
- Resume view and download option
- Contact form with basic validation
- Social media and professional profile links

---

## 📄 Pages / Sections Explanation

### 1. Home Page
The Home section is the landing area of the website. It introduces me with my name, role, and a short tagline. It gives visitors a quick first impression about who I am and what I do.

### 2. About Page
The About section provides a brief introduction about me, my background, and my interest in full stack development. It explains my passion for building web applications and learning new technologies.

### 3. Skills Page
The Skills section displays the technical skills and tools I know, such as **Python, C++, Java, HTML, CSS, JavaScript, React, MySQL, MongoDB, UI/UX, Canva, and Git**. This section highlights my technical knowledge in a clear visual format.

### 4. Projects Page
The Projects section showcases the important projects I have worked on. Each project includes a title and description. It helps visitors understand my practical experience and development skills.

Example projects included:
- Personal Portfolio Website
- Travel Booking Website
- Smart Fraud Detection in Online Payment

### 5. Certificates Page
The Certifications section displays the certificates I have earned from different training programs and courses. It shows my learning progress and commitment to improving my technical skills.

### 6. Internships Section
This section highlights my internship training experiences, such as **Full Stack Web Development** and **AI with Python**. It explains the practical exposure I gained during training.

### 7. Resume Page
The Resume section allows visitors to view and download my resume in PDF format. This makes it easy for recruiters or employers to access my professional details.

### 8. Contact Page
The Contact section includes a contact form where users can enter their name, email, subject, and message. It also contains my email, phone number, LinkedIn, GitHub, and social media links for direct communication.

## 🛠️ Technologies Used

- **HTML5** – For structuring the website
- **CSS3** – For styling and responsive design
- **JavaScript** – For interactivity and dynamic features

## ⚙️ Functionalities Implemented

- Theme switching using JavaScript and localStorage
- Mobile navigation menu
- Scroll-based active navigation tracking
- Reveal animations using Intersection Observer
- Project category filtering
- Contact form validation
- Resume file availability check
